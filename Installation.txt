********************************************
*****       Manual installation        *****
********************************************
1- Right click on RK-Vin-Base.zip and choose "Extract to "RK-Vin-Base".
You're supposed to have 2 folders: Extras and Remake Modules

2- In a new windows, open LifePlay game folder: ...\LifePlay\Content\Modules\.
And highlight these folders (all folders beginning with vin_):

    \Modules\vin_Base
    \Modules\vin_BDSM
    \Modules\vin_Corruption
    \Modules\vin_Interracial
    \Modules\vin_Netori
    \Modules\vin_OldYoung
    \Modules\vin_NTR
    \Modules\vin_Polyamorous
    \Modules\vin_Pregnancy
    \Modules\vin_Tentacles

3- Always inside: ...\LifePlay\Content\Modules\, delete or move these folders (all folders beginning with vin_) to an another places on your pc for backup.

4- Open RK-Vin-Base folder and search for the folder called "Remake Modules".
And highlight these folders:

    \Modules\rk_vin_Base
    \Modules\vin_Base
    \Modules\vin_BDSM
    \Modules\vin_Corruption
    \Modules\vin_Interracial
    \Modules\vin_Netori
    \Modules\vin_OldYoung
    \Modules\vin_NTR
    \Modules\vin_Polyamorous
    \Modules\vin_Pregnancy
    \Modules\vin_Tentacles

5- Copy/move the highlighted folders, to: ...\LifePlay\Content\Modules\

6- Make sure you don't leave an extra directory level for the mods.

7- After the mod have been installed, make sure the mods <u>rk_vin_Base</u> and <u>vin_Base</u> are activated in the Mod Manager.
Ctrl + M: Open mod manager

8- Finally, activate any Mods compatible with LP Remake you want.

9- If you find any bug let me know, I will try to fix it ASAP.

'''Question:''' What about the other files (.gitattributes, LICENSE, README.md,... )?
'''Answer:''' You don't need it!



'''Question:''' Do I really need to install these mods too?
    \Modules\vin_BDSM
